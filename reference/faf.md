# FAF Token Commands

## faf

FAF staking dashboard: staked balance, rewards, tier, and revenue share.

**Aliases:** `faf status`

## faf stake

Stake FAF tokens.

```bash
faf stake <amount>
```

## faf unstake

Unstake FAF tokens (creates an unstake request).

```bash
faf unstake <amount>
```

## faf claim

Claim FAF rewards and USDC revenue share.

**Aliases:** `faf claim rewards`, `faf claim revenue`, `faf claim rebate`

## faf tier

View VIP tier levels, requirements, and fee discount benefits.

**Aliases:** `faf tiers`, `faf vip`, `faf levels`

## faf rewards

View pending FAF rewards and USDC revenue share breakdown.

## faf points

Voltage points tier and multiplier status.

**Aliases:** `faf voltage`

## faf referral

Referral program status and rebate earnings.

**Aliases:** `faf referrals`

## faf requests

View pending unstake requests.

**Aliases:** `faf unstake requests`, `faf pending`

## faf cancel

Cancel a pending unstake request by index.

```bash
faf cancel <index>
```
